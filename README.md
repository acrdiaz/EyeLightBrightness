# EyeLightBrightness

## Project Description
EyeLightBrightness is a C# application designed to control the brightness of eye lights. It provides an intuitive interface for users to adjust and customize their lighting settings according to their preferences, ensuring optimal visual comfort and reducing eye strain.

## Features
- User-friendly interface for adjusting brightness
- Real-time changes in lighting settings
- Customization options to suit individual needs

## Installation
1. Clone the repository: `git clone https://github.com/acrdiaz/EyeLightBrightness`
2. Open the project in your preferred C# IDE.
3. Build and run the application.

## Usage
- Launch the application and use the sliders/controls to adjust brightness settings.
